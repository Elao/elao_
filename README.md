# elao_

The Elao website.

## Prerequisite

Install `cecil` executable:

```
curl -LO https://cecil.app/cecil.phar
mv cecil.phar /usr/local/bin/cecil
chmod +x /usr/local/bin/cecil
```

### Install npm dependencies

    npm install

## Contributing

### Start the dev server

    cecil serve --drafts

### Start asset watcher

    npm run watch

## Build

    npm run build
    cecil build
